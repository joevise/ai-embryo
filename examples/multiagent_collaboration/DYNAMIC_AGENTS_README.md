# 动态多智能体协作系统

🤖 **AI驱动的智能体动态生成和协作系统**

基于FuturEmbryo框架的新一代多智能体系统，支持根据任务需求自动设计和创建专门的智能体。

## 🚀 快速启动

### 1. 进入项目目录
```bash
cd /Users/dajoe/joe_ai_lab/FuturEmbryo/FuturEmbryo-v2.1c_claude/examples/multiagent_collaboration
```

### 2. 配置API密钥
在 `dynamic_multiagent_demo.py` 文件中，修改以下行：
```python
setup_futurx_api(
    api_key="your-actual-api-key",  # 替换为您的真实API密钥
    base_url="https://litellm.futurx.cc"
)
```

### 3. 启动系统
```bash
# 方法1: 使用启动脚本（推荐）
./run_dynamic_demo.sh

# 方法2: 直接运行Python
python dynamic_multiagent_demo.py
```

## 🎯 新功能特性

### 🧠 AI驱动的智能体设计
- **任务分析**: 智能分析用户任务的性质、复杂度和所需技能
- **智能体设计**: 自动设计专门的AI智能体，包括角色、能力和工具配置
- **动态创建**: 运行时创建和配置智能体，无需预定义

### 🔧 智能工具配置
- **自动配置**: 根据任务需求自动为智能体配置合适的工具
- **工具类别**: 支持文件操作、网络搜索、数据分析、文本处理等
- **可扩展性**: 可以轻松添加新的工具类别和功能

### 📋 三阶段工作流
1. **分析设计阶段**: 分析任务并设计智能体
2. **确认阶段**: 用户确认执行计划
3. **执行阶段**: 创建智能体并协作完成任务

## 🏗️ 系统架构

### 核心组件

1. **CoordinatorAgent** (`agents/coordinator_agent.py`)
   - 智能任务分析和协调
   - AI驱动的智能体设计
   - 三阶段工作流管理

2. **DynamicAgentFactory** (`core/dynamic_agent_factory.py`)
   - 动态智能体创建和配置
   - 工具能力自动分配
   - 资源管理和清理

3. **UserAwareAgent** (`agents/user_aware_agent_v2.py`)
   - 基础智能体实现
   - 工具调用和记忆管理
   - 组合模式设计

4. **ContextBuilder** (`core/context_builder.py`)
   - 统一的上下文管理
   - 记忆和工具集成
   - 用户偏好处理

### 工作流程

```
用户输入任务
    ↓
协调器分析任务
    ↓
设计专门智能体
    ↓
用户确认计划
    ↓
动态创建智能体
    ↓
协作执行任务
    ↓
整理输出结果
```

## 🔧 与原系统的区别

### 原系统 (静态智能体)
- 预定义固定的智能体角色
- 手动配置工具和能力
- 用户直接选择智能体

### 新系统 (动态智能体)
- AI分析任务并设计智能体
- 自动配置工具和能力
- 协调器智能分配工作
